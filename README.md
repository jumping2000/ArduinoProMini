# ArduinoProMini
Arduino Pro Mini board for MySensors

The PCB is designed as Mysensors node 

**Supply**

LiOn battery or 5V USB

**Boards features:**
1. LiOn battery support
2. NRF24L01+ support
3. RFM69 support
