# ArduinoProMini
Arduino Pro Mini board for MySensors,the board is designed for multiple sensors.

**Supply**

LiOn battery or 5V USB

**Boards features:**
1. LiOn battery support
2. NRF24L01+ support
3. RFM69 support
4. designed for multiple sensors
